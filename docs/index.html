Schritt-für-Schritt-Anleitung zur Erstellung einer Angular-Anwendung

Voraussetzungen

  • Node.js: Installieren Sie Node.js von nodejs.org. Überprüfen Sie die Installation mit: 
    node -v
npm -v
  • Angular CLI: Installieren Sie Angular CLI mit: 
    npm install -g @angular/cli

Schritt 1: Eine neue Angular-Anwendung erstellen

  1. Öffnen Sie das Terminal und navigieren Sie zu dem gewünschten Verzeichnis.
  2. Erstellen Sie ein neues Projekt: 
    ng new todo-app
  3. Beantworten Sie die Fragen des Angular CLI (z.B. CSS als Stil).

Schritt 2: Die Anwendung starten

  1. Navigieren Sie in das neu erstellte Verzeichnis: 
    cd todo-app
  2. Starten Sie die Anwendung: 
    ng serve
  3. Öffnen Sie einen Browser und gehen Sie zu http://localhost:4200.

Schritt 3: HTML und CSS hinzufügen

  1. Öffnen Sie src/app/app.component.html und fügen Sie folgendes HTML hinzu: 
    <div class="container">
  <form>
    <div class="form-group">
      <h1 class="text-center text-primary">Todo App</h1>
      <div class="input-group-prepend">
        <input type="text" class="form-control" placeholder="Add Todo" name="todo">
        <span class="input-group-text">Add</span>
      </div>
    </div>
  </form>
</div>
  2. Öffnen Sie src/app/app.component.css und fügen Sie folgendes CSS hinzu: 
    body { padding: 0; margin: 0; }
form { max-width: 25em; margin: 1em auto; }

Schritt 4: Daten erfassen und anzeigen

  1. Bearbeiten Sie src/app/app.component.ts, um ein Array zu erstellen und Eingaben zu erfassen: 
    export class AppComponent {
  todoArray = [];
  addTodo(value: string) {
    if (value) this.todoArray.push(value);
  }
}
  2. Aktualisieren Sie app.component.html, um die Eingaben zu erfassen und anzuzeigen: 
    <div class="container">
  <form (ngSubmit)="addTodo()">
    <div class="form-group">
      <h1 class="text-center text-primary">Todo App</h1>
      <div class="input-group-prepend">
        <input type="text" #todoInput class="form-control" placeholder="Add Todo" (ngModel)="todoInput.value">
        <span class="input-group-text" (click)="addTodo(todoInput.value)">Add</span>
      </div>
    </div>
    <ul>
      <li *ngFor="let todo of todoArray">{{ todo }}</li>
    </ul>
  </form>
</div>

Schritt 5: Löschen von Einträgen

  1. Fügen Sie eine Methode zum Löschen hinzu in app.component.ts: 
    deleteItem(todo: string) {
  this.todoArray = this.todoArray.filter(item => item !== todo);
}
  2. Aktualisieren Sie app.component.html: 
    <li *ngFor="let todo of todoArray">
  {{ todo }}
  <i class="material-icons" (click)="deleteItem(todo)">delete</i>
</li>
